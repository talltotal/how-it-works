# babel-plugin-danger-remove-unused-import

删除文件中未使用的import（有些import除了export，还是有执行一些内容，所以删除存在风险）

  • 定义数据储存对象runtimeData
    • 在pre定义，在post删除
  • 在ImportDeclarationpath以变量名为key存储path
  • 在JSXIdentifier/Identifierpath删除使用的变量path
  • runtimeData中残留的即未使用的import，做删除操作
