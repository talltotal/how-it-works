# table

主要为验证是否为标准的table，解析table行列的position

  1. 至少两行
  2. offset [0,4) -- 大于等于4的按code处理
  3. 第二行满足正则/^[-:|][-:|\s]*$/
  4. 确定 columns
    • 按第二行的 |
    • 允许前后空白，不允许中间空白；如|---|,--||--
    • 前后空白不视为一列
    • align
      • 最后一个字符是:
        • 第一个字符是: - center
        • right
      • 第一个字符是: - left
